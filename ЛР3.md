    Deactivate Client

title: Свайкин ЛР3 №2

sequenceDiagram

    actor User as User

    participant Browser as Браузер

    participant Server as Сервер сайта

    participant SMSService as SMS-Сервис

    participant Mobile as Мобильное устройство

    User->>Browser: Вводит логин и пароль

    Browser->>Server: Отправляет логин и пароль

    Activate Server

    Server->Browser: Ответ: «Пароль верный, требуется 2FA»

    Deactivate Server

    Browser->User: Запрос на ввод кода 2FA

    Note right of Browser: Пользователь ожидает SMS

    Server->>SMSService: Запрос на отправку SMS с кодом

    Activate SMSService

    SMSService->>Mobile: Отправляет SMS с кодом подтверждения

    SMSService->Server: Подтверждение отправки

    Deactivate SMSService

    Mobile->User: Получение SMS с кодом

    User->>Browser: Вводит код из SMS

    Browser->>Server: Отправляет код подтверждения

    Activate Server

    Server->Browser: Подтверждение успешной авторизации

    Deactivate Server

    Browser->User: Перенаправление в личный кабинет

title: Свайкин ЛР3 №3
---
sequenceDiagram
    actor Buyer as Покупатель
    participant Cashier as Кассир
    participant POS as Кассовый аппарат (POS-система)
    participant PaymentTerminal as Платежный терминал
    Buyer->>Cashier: Передает товары для сканирования
    activate Cashier
    loop Для каждого товара
        Cashier->>POS: Сканирует штрихкод товара
        activate POS
        POS-->>Cashier: Отображает название и цену
        deactivate POS
    end
    Cashier->>POS: Нажимает "Итого"
    activate POS
    POS-->>Cashier: Показывает итоговую сумму
    deactivate POS
    Cashier-->>Buyer: Озвучивает сумму к оплате
    Buyer->>Cashier: Передает банковскую карту
    Cashier->>PaymentTerminal: Подносит карту к терминалу
    activate PaymentTerminal
    PaymentTerminal-->>Cashier: Запрос подтверждения суммы
    deactivate PaymentTerminal
    Cashier->>POS: Подтверждает оплату
    activate POS
    POS->>PaymentTerminal: Команда на списание средств
    activate PaymentTerminal
    PaymentTerminal-->>POS: Подтверждение успешной оплаты
    deactivate PaymentTerminal
    POS->>POS: Печать чека
    POS-->>Cashier: Сигнал об успешной операции
    deactivate POS
    Cashier->>Cashier: Упаковывает товары
    Cashier->>Buyer: Передает чек и упакованные товары
    Cashier-->>Buyer: Говорит "Спасибо за покупку!"
    deactivate Cashier
